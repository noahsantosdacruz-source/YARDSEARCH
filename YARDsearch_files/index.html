Un frontend React léger, pensé pour se brancher progressivement sur une API Laravel.

Frontend

React 18 + Vite

SPA sans routeur externe, state local pour la navigation.

Gratuit

Laravel + Sanctum

Authentification par token, API REST/GraphQL.

Gratuit

PostgreSQL / Supabase

Gratuit

Prochaines étapes

Branchement API

Ordre conseillé pour passer du mock au réel.

01

Authentification

OAuth2 / Sanctum, token stocké côté client.

02

Listing pros

GET /api/v1/pros à la place de mockData.pros.

03

Création de devis

POST /api/v1/devis depuis Devis.jsx.

04

Messagerie temps réel

WebSockets (Laravel Echo) ou MQTT (HiveMQ).

05

Géolocalisation

GET /api/v1/bons-plans?lat&lng pour les bons plans à proximité.
